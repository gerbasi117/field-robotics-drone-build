# GitHub Polish Checklist

Before sharing this repo with recruiters:

- [ ] README clearly explains the project in the first 5 lines.
- [ ] Parts list is filled with actual components.
- [ ] Power architecture is clear.
- [ ] Wiring diagram is readable.
- [ ] Telemetry plan explains ground-station connection.
- [ ] Safety checklist is complete.
- [ ] Bench-test plan is complete.
- [ ] Risk register is complete.
- [ ] Real photos are added under `images/`.
- [ ] No private information is included.
- [ ] Resume bullets are copied into the main resume.
