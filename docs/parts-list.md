# Parts List

## Flight control and avionics

| Subsystem | Component | Notes |
|---|---|---|
| Flight controller | ModalAI Flight Core V2 | Primary autopilot/flight-control computer |
| GPS | ModalAI u-blox M10 GPS | Navigation and position source |
| Telemetry | 915MHz telemetry radio pair | Ground-station communication |
| Receiver/control | ExpressLRS 915MHz system | Manual control link |
| Power monitoring | Power Module V3 | Voltage/current monitoring and FC power |

## Propulsion

| Subsystem | Component | Notes |
|---|---|---|
| ESC | Holybro Tekko32 F4 Metal 4-in-1 65A, 6S | Four motor outputs, high-current propulsion control |
| Motors | 4x EMAX ECO II 2207 1700KV | Brushless motors |
| Propellers | HQProp DP 7x3.5x3 tri-blade | Large tri-blade props for thrust testing |

## Power system

| Subsystem | Component | Notes |
|---|---|---|
| Batteries | 2x CNHL 9500mAh 6S 22.2V packs | Parallel setup for capacity/current headroom |
| Main connector | XT90 parallel harness with XT90-S anti-spark | Main high-current connection |
| Wiring | 12 AWG / 14 AWG / 16 AWG as appropriate | High-current and shorter distribution runs |
| Distribution | Matek XT60 PDB / power distribution | Used for power routing and accessories |
